My Personal Web Page

Framework: Jekyll
Front-end: Foundation5, Classic CSS, and SVG

My personal website is my second official web project I have created. As a static framework for the site, I used Jekyll (a Ruby gemfile). I also used Foundation5, CSS, and SVG to make things aesthetically pleasing. Additionally, I made a photo-Carto-DB feature on my photos page which circumvents some of the liquid template mark-up Jekyll uses in order to make use of CartoDB's own mark-up language and make the map more interactive.
